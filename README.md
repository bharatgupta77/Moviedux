# Movie Browser Application

This project is a Movie Browser application that allows users to explore a collection of movies, search by title, filter
by genre and rating, and view details for each movie. The app is built using React and demonstrates interactive
front-end functionality.

## Features

- Search movies by titles using a search bar.
- Filter movies by genre (e.g., Action, Fantasy, Drama, etc.).
- Filter movies based on user ratings (Good, Okay, or Bad).
- Responsive movie card components displaying the title, genre, and rating of movies.

## Data Source

Movie information is sourced from a static JSON file (`movies.json`) stored in the `public` directory. The file contains
details such as movie titles, genres, ratings, images, and unique IDs.

## Components Used

- `MoviesGrid` dynamically renders the movie collection with filters applied.
- `MovieCard` displays individual movie details in a styled card format.

Start exploring the movies by running the app locally with your preferred development setup.

## Running and Building the Application

To run the application in development mode:

1. Install the dependencies using:
   ```
   npm install
   ```

2. Start the application using:
   ```
   npm start
   ```
